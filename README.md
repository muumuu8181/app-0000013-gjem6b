# 🎮 NEON TETRIS - めちゃくちゃ格好良いテトリス

## 概要
ネオンに輝く究極のテトリス体験。美しいビジュアルエフェクト、パワーアップシステム、そして滑らかなゲームプレイが特徴の次世代テトリスゲームです。

## 🎯 ゲーム特徴

### ✨ 主要機能
- **7種類のテトロミノ**: 全ての標準テトリスピース
- **高度な回転システム**: ウォールキック対応
- **ホールド機能**: 戦略的なピース管理
- **ゴーストピース**: 落下位置プレビュー
- **レベルシステム**: 段階的な難易度上昇

### 💫 パワーアップ
1. **💣 ボム**: 5x5範囲のブロックを破壊
2. **⏱️ タイムスロー**: ゲーム速度を50%に減速
3. **⚡ レーザー**: 最下段の不完全な行を消去
4. **👻 ゴースト**: 10秒間ブロックが透過

### 🏆 実績システム
- 🌟 **First Line**: 初めてのライン消去
- 🏆 **Tetris!**: 4ライン同時消去
- 🔄 **T-Spin Master**: T-スピン成功
- 👑 **Combo King**: 10コンボ達成

## 🎮 操作方法

### キーボード
- **←→**: 左右移動
- **↓**: ソフトドロップ
- **Space**: ハードドロップ
- **↑/Z**: 時計回り回転
- **X**: 反時計回り回転
- **C**: ホールド

### モバイル
専用のタッチコントロールUIで快適操作
